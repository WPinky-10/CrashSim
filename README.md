# CrashSim

CrashSim is a Python-based simulation project that uses external libraries such as NumPy and Matplotlib to perform calculations, simulations, and visualizations.

---

## Requirements

* Python 3.8 or higher

### Dependencies

The project requires the following Python libraries:

* `numpy`
* `matplotlib`
* `openpyxl`
* `tkinter*`

> Note:
> `tkinter` is included with most Python installations on Windows and macOS.
> On Linux, you may need to install it manually:

```bash
sudo apt install python3-tk
```

---

## Installation

### Option 1 (Recommended)

Run the installer script:

```bash
installer.bat
```

### Option 2 (Manual Setup)

1. Download and extract the project folder
2. Open a terminal (Command Prompt / PowerShell) inside the project folder
3. Install dependencies:

```bash
pip install -r requirements.txt
```

### If that fails:

```bash
pip install numpy matplotlib
```

---

## How to Run

### Option 1 (Recommended)

```bash
run.bat
```

### Option 2 (Manual Run)

```bash
python CrashSim.py
```

If that does not work:

```bash
python3 CrashSim.py
```

---

## Troubleshooting

### ModuleNotFoundError

Install missing dependencies:

```bash
pip install numpy matplotlib
```

### Python not recognized

Make sure Python is installed and added to your system PATH.

Check with:

```bash
python --version
```
