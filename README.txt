CrashSim

CrashSim is a Python-based simulation project that uses external libraries such as NumPy and Matplotlib to perform calculations, simulations, and visualizations.

Requirements

Make sure you have Python 3.8 or higher installed.

This project depends on the following Python libraries:
- numpy
- matplotlib
- openpyxl
- tkinter*

*Tkinter is included with most Python installations on Windows and macOS.
On Linux, you may need to install it manually:
sudo apt install python3-tk


Installation
Use installer.bat
or
1. Download and extract the project folder.
2. Open a terminal (Command Prompt / PowerShell) inside the project folder.
3. Install the required dependencies:
pip install -r requirements.txt

If that does not work, install manually:
pip install numpy matplotlib


How to Run
Use run.bat
or
Run the program using:
python CrashSim.py
If that does not work, try:
python3 CrashSim.py


Troubleshooting

ModuleNotFoundError
If you get missing module errors, install dependencies:
pip install numpy matplotlib

Python not recognized
Make sure Python is installed and added to PATH.

Check with:
python --version
